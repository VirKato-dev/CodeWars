Для чего используется SequenceInputStream?
Класс SequenceInputStream позволяет сливать вместе несколько экземпляров класса InputStream.
Конструктор принимает в качестве аргумента либо пару объектов класса InputStream, либо интерфейс Enumeration.
Во время работы класс выполняет запросы на чтение из первого объекта класса InputStream и до конца,
а затем переключается на второй. При использовании интерфейса работа продолжится по всем объектам класса InputStream.
По достижении конца, связанный с ним поток закрывается.

Закрытие потока, созданного объектом класса SequenceInputStream,
приводит к закрытию всех открытых потоков.
